# Task 3: Exploratory Data Analysis (EDA)

## Dataset
Netflix Movies and TV Shows

## Objective
The objective of this task is to perform Exploratory Data Analysis (EDA) to understand data patterns, trends, and feature behavior using visualizations.

## Tool Used
Python (Google Colab)

## Libraries Used
- Pandas
- Matplotlib

## Dataset Description
The dataset contains information about Netflix content including Movies and TV Shows, with attributes such as type, country, release year, rating, and duration.
Total Records: 8807
Total Features: 12

## Analysis Performed
- Data loading and basic understanding
- Handling missing values
- Categorical analysis (Movies vs TV Shows, Ratings)
- Numerical distribution analysis (Release Year)
- Outlier detection using box plots
- Country-wise content analysis
- Identification of important features

## Key Insights
- Netflix has more movies than TV shows.
- Most content was released after 2010.
- TV-MA and TV-14 are the most common ratings.
- The United States produces the highest number of Netflix titles.
- Few older titles appear as outliers.

## Conclusion
Exploratory Data Analysis helped in understanding Netflix content distribution, trends, and feature behavior. The insights gained from EDA are useful for further predictive modeling and business decisions.
